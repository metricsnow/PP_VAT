# Agent Creator Command

## Command Activation
**Command**: `/agent-creator`

## Purpose
Forces activation of Agent Creator role for agent creation, optimization, and meta-agent management.

## Agent Instructions
Reference: `development_framework_v2/framework/agents/agent-creator_agent.md`

## Usage
Type `/agent-creator` followed by your instruction or question.

## What it does
- Activates Agent Creator role
- Applies agent creation methodology
- Creates and optimizes agent definitions
- Manages meta-agent operations
- Designs agent workflows
- Optimizes agent performance
- Ensures agent quality standards

## Example
`/agent-creator Create a new agent for handling API integration tasks`
